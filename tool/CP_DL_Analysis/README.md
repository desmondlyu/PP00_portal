# 🔬 CP Wafer Map & DL Analysis (完全離線單網頁版)

這是一個專為半導體測試數據分析設計的網頁工具。100% 離線執行，無須連網或安裝任何伺服器環境。

---

## 🌟 核心功能特色

### 1. 100% 本地離線運作
* **數據安全無虞**：所有運算、資料解析和圖表渲染完全在您的本機瀏覽器中完成，絕不會將任何 wafer 資料傳送到外部網路。

### 2. 雙向框選聯動功能 (關鍵特色)
* **晶圓空間框選（Spatial Filter）**：在晶圓圖上按住滑鼠左鍵並拖曳拉出藍色虛線框，可以篩選出特定區域的 Die。其餘三張圖表（相關性、直方圖、箱形圖）會自動**同步更新**，只分析該框選區域內的資料。點擊右上角的紅色「清除空間篩選」按鈕即可恢復。
* **圖表高亮同步（Highlight Sync）**：在相關性散佈圖、直方圖或箱形圖上拖曳拉框，晶圓圖上對應的點位會立即同步顯示**紅色外框**，方便您快速追蹤異常點在晶圓上的物理分佈。

### 4. 多維度數據統計圖表
* **相關性分析 (Correlation)**：分析兩個測試參數之間的關聯性，自動計算 Pearson R 相關係數，可切換對數尺度 (Log Scale) 並支援顯示線性趨勢線。
* **分佈箱形圖 (Boxplot)**：展示多個選定參數的統計數據（最大值、最小值、中位數及四分位數），利於進行參數間的分佈比對。
* **頻率直方圖 (Histogram)**：展示單一或多個參數的數值頻率分佈，支援 X/Y 軸對數尺度切換。

## 🛠️ 常見問題排解

* **Q：為什麼上傳檔案後圖表是空的？**
  * 請檢查 DL 檔案格式是否正確。特別是第 6 行是否包含 `<X>` 與 `<Y>` 欄位，以及資料行是否與欄位數量對齊。
