## 使用方式

1. **開啟頁面** - 進入預約系統主畫面
2. **選擇日期** - 點選您要預約的日期
3. **選擇測試機台** - 從可用的機台清單或平面圖中選擇
4. **新增預約** - 點「新增預約」按鈕
5. **填寫預約資訊** - 輸入以下資訊：
   - 姓名（必填）
   - 部門（必填）
   - 開始時間
   - 結束時間
6. **確認送出** - 送出後系統會立即更新畫面

## 預約規則

- 同一台機台在同一時段不可重疊預約。
- 若預約時段重疊，按下「確定」會跳出警告視窗提示。
- 結束時間必須晚於開始時間。
- 部門與姓名為必填欄位。
- 刪除預約：僅能刪除由同一台電腦建立的資料。
- 編輯預約：僅能編輯由同一台電腦建立的資料。

## 檢視方式

系統提供兩種檢視模式：
- **卡片模式** - 清單式顯示所有預約
- **平面圖模式** - 空間分布圖顯示機台位置
