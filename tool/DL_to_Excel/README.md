# DL → Excel 轉換工具使用說明書 📂✨

本工具是一個專門為 CP Datalog 純文字檔案設計的極速 Excel 轉換網頁。

採用**完全本機離線運作模式**，所有檔案解析都在您的瀏覽器內完成，資料絕不上傳，徹底保障機密數據的隱私安全。

---

## 🚀 快速開啟方式

1. 開啟工具資料夾 `DL_to_Excel`。
2. 尋找 **`index.html`** 檔案。
3. **按兩下（雙擊）** `index.html`，即可直接在瀏覽器中開啟轉換介面，無需安裝任何軟體。

---

## 📖 使用者操作教學

### 步驟 1：載入 Datalog 檔案
*   將您要轉換的 Datalog 檔案（支援多選）直接**拖曳**到網頁中央的灰色虛線框中。
*   或者點選 **「選擇檔案」** 按鈕，在電腦中選取您的檔案。

### 步驟 2：設定匯出模式與挑選欄位
檔案載入成功後，網頁會自動跳轉至設定畫面：

1.  **選擇匯出方式**：
    *   **合併成一個 Excel**：將多個上傳檔案的資料合併到同一個 Excel 工作表中。為方便識別，Excel 中會自動在 Column A 新增一欄 `Source`，標記每筆資料來自哪一個原始檔名。
    *   **分開儲存並打包**：每個檔案各別轉換成獨立的 Excel，系統會自動將所有 Excel 壓縮成一個 **ZIP 壓縮檔** 下載，只需下載一次即可解壓使用。
2.  **挑選匯出欄位**：
    *   在欄位選擇區中，**勾選**您想要保留的欄位名稱。
    *   使用 **「搜尋欄位...」** 快速輸入關鍵字過濾特定欄位。
    *   提供 **「全選」** 與 **「取消全選」** 的快速鍵。
    *   *提示：在「合併模式」下，`Source`（來源檔名）為強制勾選欄位，無法取消。*

### 步驟 3：開始轉換與下載
*   確認模式與欄位後，點選右下角的 **「確認匯出」**（或按鍵盤 **Enter** 鍵）。
*   系統會以極快速度處理完畢並**自動觸發瀏覽器下載**。同時，網頁會顯示該次轉換的統計數據（包含檔案數、總資料列數、解析耗時與寫入耗時）。

## 🛑 Fail-Safe 欄位防呆機制說明

當您選擇 **「合併成一個 Excel」** 模式時，如果上傳的多個 Datalog 檔案中，有任何一個檔案的欄位結構與基準檔案不符（例如欄位數量不同、缺少欄位、或多了新欄位）：
1.  系統會**立即中斷轉換**。
2.  網頁底部的狀態列會以**紅色字體**詳細顯示哪一個檔案有何處不一致。
3.  **「選擇欄位並匯出」按鈕會自動鎖死**，防止您合併出結構錯亂的損毀 Excel。
4.  *排除方法：請確認所有要合併的 Datalog 格式一致後再重新上傳；或者將匯出方式切換成「分開儲存」，即可正常各別匯出。*

## 🧪 快速上手測試

專案已貼心附帶三個測試 sample 檔案，存放於 **`samples/`** 資料夾中，供您體驗：
1.  **`samples/sample1.log`** & **`samples/sample2.log`**：兩個欄位結構完全相同的檔案。
    *   *測試方法*：同時將這兩個檔案拖入網頁，選擇「合併」並匯出，即可下載合併後的 Excel。
2.  **`samples/sample_bad.log`**：一個欄位結構不同的錯誤示範檔案。
    *   *測試方法*：同時拖入 `samples/sample1.log` 與 `samples/sample_bad.log`，選擇「合併」，您將會在下方狀態列看到紅色警告訊息且按鈕無法點擊，這即是防呆機制的正常反應。
